# Habilidades e Ferramentas Detalhadas

## Habilidades

### Backend
- **Python**: Desenvolvimento de APIs, automação e scripts.
- **C**: Programação de sistemas de baixo nível e otimização de desempenho.
- **C++**: Desenvolvimento de sistemas de alto desempenho e jogos.
- **Java**: Desenvolvimento de software corporativo e aplicativos móveis.
- **Lua**: Criação de scripts para jogos e automações simples.
- **Node.js**: Desenvolvimento de servidores e aplicações backend.

### Frontend
- **HTML**: Criação de estrutura de páginas web.
- **CSS**: Estilização de páginas web, incluindo Flexbox, Grid e pré-processadores como SASS.
- **JavaScript**: Desenvolvimento de interatividade em páginas web, utilizando frameworks como React e Vue.js.

### Cibersegurança
- **Análise de redes**: Uso de ferramentas como Wireshark e Nmap para inspecionar pacotes e segurança de redes.
- **Testes de penetração**: Uso de ferramentas como Metasploit para explorar vulnerabilidades de sistemas.
- **Criptografia**: Compreensão dos fundamentos de criptografia para garantir a segurança de dados.

## Ferramentas

### Ferramentas de Desenvolvimento
- **Git**: Controle de versão para gerenciar o código e colaborar com equipes.
- **GitHub**: Plataforma de hospedagem de código e colaboração em projetos.
- **WebStorm**: IDE especializada para o desenvolvimento frontend com JavaScript, Node.js e frameworks modernos.
- **Figma**: Ferramenta de design de interfaces e prototipagem.
- **VMware Workstation**: Ferramenta de virtualização para testar diferentes sistemas operacionais e ambientes isolados.
- **Docker**: Plataforma para criação, deploy e execução de aplicativos em contêineres.

### Ferramentas de Cibersegurança
- **Kali Linux**: Sistema operacional focado em testes de penetração e auditoria de segurança.
- **Metasploit**: Framework para desenvolver e executar exploits em sistemas de teste.
- **Wireshark**: Ferramenta para captura e análise de pacotes de rede.
- **Nmap**: Scanner de rede para auditoria de segurança.

## Sistemas Operacionais
- **Linux**: Usado para desenvolvimento e cibersegurança, com distribuições como Kali Linux, Ubuntu, Arch, entre outras.
- **Windows**: Usado para o desenvolvimento de software no ecossistema Microsoft, além de suporte a jogos.
